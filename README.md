## AnonGT Commands

___

### print banner help

> ### sudo anongt

### Anonymous Mode Start

> ### sudo anongt start

### Anonymous Mode Start With Secure Tor Bridges

> ### sudo anongt start+

### Anonymous Mode Stop

> ### sudo anongt stop

### Watch Tor Traffic

> ### sudo anongt status

### Get Your #IP Address

> ### sudo anongt myip

### Change Tor Identity

> ### sudo anongt chngid

### Change #IP Automatically
> ### sudo anongt autochng

### Anti MITM
> ### sudo anongt antimitm

### Change Mac Address For All Interfaces

> ### sudo anongt chngmac

### Revert Mac Address For All Interfaces

> ### sudo anongt rvmac

### Onion Links Generator
> ### sudo anongt oniongen

### Onion Links Checker

> ### sudo anongt checko

### Anonymous Share Files
> ### sudo anongt share

### Anonymous Receive Files
> ### sudo anongt receive

### Anonymous Chat
> ### sudo anongt chat

### Host A Website
> ### sudo anongt website

### Memory Wipe & Clear Logs
> ### sudo anongt wipe

### If Shutdown Without Stop
> ### sudo anongt fix

### AnonGT Check Update

> ### sudo anongt checku

### AnonGT About

> ### sudo anongt about

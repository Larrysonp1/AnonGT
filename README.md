## AnonGT Commands

___

### print banner help

> ### sudo anongt

### Anonymous Mode Start

> ### sudo anongt start

### Anonymous Mode Start With Secure Tor Bridges

> ### sudo anongt start+

### Anonymous Mode Stop

> ### sudo anongt stop

### Watch Tor Traffic

> ### sudo anongt status

### Get Your Public Information

> ### sudo anongt myinfo

### Anonymous Mode Change Tor IP

> ### sudo anongt chngid

### Anti MITM
> ### sudo anongt amitm

### Change Mac Address For All Interfaces

> ### sudo anongt chngmac

### Revert Mac Address For All Interfaces

> ### sudo anongt rvmac

### Onion Links Generator
> ### sudo anongt oniongen

### Onion Links Checker

> ### sudo anongt checko

### Clear Configs & Logs

> ### sudo anongt wipe

### AnonGT Check Update

> ### sudo anongt checku

### AnonGT About

> ### sudo anongt about

## Change Tor Bridge Dir
> ### https://gitlab.torproject.org/tpo/anti-censorship/team/-/wikis/Default-Bridges
> ### /etc/tor/torrc

> ### /usr/share/AnonGT/core/config/config.py
